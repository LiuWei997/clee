# CLEE

A fast, interactive command-line interface (CLI) built with Python and [Typer](https://tiangolo.com) to manage your LeetCode problems. Pull challenges locally, write solutions in your favorite editor, and push your code directly back to LeetCode right from your terminal.

## ✨ Features

- **📥 Pull Problems**: Download problem descriptions, difficulty stats, and code templates by ID or URL slug.
- **📤 Push Solutions**: Submit your code directly to LeetCode and see execution results (Pass/Fail, Runtime, Memory) instantly.
